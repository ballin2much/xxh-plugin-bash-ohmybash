## Install
From xxh repo:
```
xxh +I xxh-plugin-bash-ohmybash
xxh yourhost +s bash +if
```
From any repo:
```
xxh +I xxh-plugin-bash-ohmybash+git+https://github.com/xxh/xxh-plugin-bash-ohmybash
```    
Connect:
``````
xxh yourhost +s bash +if
```
